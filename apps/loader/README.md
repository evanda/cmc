# @cmc/loader

**Status: placeholder (Phase 0).** The standalone map-authoring tool is built in
**Phase 2** (plan §10): a small React + MapLibre admin app to upload/georeference
satellite imagery, digitize building footprints and POIs, and place
georeferenced interior floor overlays organized by `level`. It shares
`@cmc/shared` types so its exports drop straight into the DB. Not shipped to end
users. For now this workspace only holds a stub to keep the monorepo green.
